## Installation

You need to install the icon wrapper components to use the animated icons.

<CodeTabs
  codes={{
    npm: `npx shadcn@latest add "https://animate-ui.com/icon.json"`,
    pnpm: `pnpm dlx shadcn@latest add "https://animate-ui.com/icon.json"`,
    yarn: `npx shadcn@latest add "https://animate-ui.com/icon.json"`,
    bun: `bun x --bun shadcn@latest add "https://animate-ui.com/icon.json"`,
  }}
